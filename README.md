# engine
Game Engine - Currently on version MASON

## Getting Started
### Setup
1. Clone this repo:
```
git clone git@github.com:Project-Rekt/engine.git
```

2. Install dependencies within the project directory: 
```
npm install
```

### To run 
Run the webpack dev server:
```
npm run dev
```

### To build 
Run the webpack build script
```
npm run build
```
